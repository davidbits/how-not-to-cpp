# Code of Conduct

## Our Standards

This is a repository about bad C++ code. Discussions will sometimes involve
strong opinions about programming practices. That is expected and welcome.

What is not welcome:

- Personal attacks
- Harassment of any kind
- Gatekeeping — everyone was a beginner once, and the anti-patterns in this
  repo exist precisely because experienced developers write them too
- Claiming that any of the code in this repository is actually good

## On the code itself

The C++ in this repository is intentionally wrong. Pointing that out is the point.

"This code is terrible" is a compliment here. "This code is actually fine" is a
rejection criterion. Keep this distinction in mind when reviewing contributions.

## Enforcement

Issues that violate these standards should be reported by opening a GitHub issue
or contacting the maintainer directly.

Reports will be reviewed. Enforcement actions range from a request to edit a
comment to removal from the project, depending on severity.

## Attribution

This Code of Conduct is short on purpose. The longer it is, the less anyone reads it.
